# Organizador-de-arquivos
Organiza automaticamente arquivos dentro de uma pasta selecionada pelo usuário, categorizando-os em subpastas conforme sua extensão.

/Selecionado
  ├── Pdfs
  │     └── documento.pdf
  ├── Planilhas
  │     └── planilha.xlsx
  ├── Imagens
  │     └── foto.jpg
  ├── Codigos Python
  │     └── script.py
